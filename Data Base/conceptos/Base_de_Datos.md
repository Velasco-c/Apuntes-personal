# Base de Datos

Colección estructurada de datos lógicamente coherentes que representan entidades del mundo real.

## Jerarquía de Información
- **[Dato](../conceptos/Dato.md):** El nivel atómico. Valores crudos (números, strings, binarios) sin contexto ni interpretación.
- **[Información](../conceptos/Información.md):** Datos procesados, organizados y estructurados que poseen significado y permiten la toma de decisiones.

## Elementos constitutivos
- **[Entidad](../conceptos/Entidad.md):** Unidad de información significativa en el sistema (ej. Cliente, Producto).
- **Atributo:** Característica que describe a una entidad.
- **Dominio:** El conjunto de valores permitidos para un atributo específico.
