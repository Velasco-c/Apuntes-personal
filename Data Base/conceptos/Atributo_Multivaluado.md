# Atributo Multivaluado

Característica de una [Entidad](../conceptos/Entidad.md) que puede contener múltiples valores para un único registro.

- *Ejemplo:* Un 'Número de Teléfono' si un usuario tiene varios, o 'Idiomas'.
- *Nota:* En el modelo relacional, estos suelen requerir una tabla separada para evitar repetir datos (evitar listas separadas por comas).
