# Información

Conjunto organizado de [datos](../conceptos/Dato.md) procesados que poseen un significado, contexto y propósito dentro de un sistema.

## Ciclo de vida:
- **Dato (Bruto)** -> **Procesamiento** -> **Información (Contextualizada)** -> **Conocimiento (Aplicada)**.
- La información reduce la incertidumbre y sirve como base para la toma de decisiones en cualquier [SGBD](../conceptos/SGBD.md).
