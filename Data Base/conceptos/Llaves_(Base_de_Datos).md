# Llaves (Base de Datos)

Mecanismos de integridad que permiten identificar, relacionar y restringir los registros dentro de una estructura relacional.

## Tipos principales:
- **[Llave Primaria](../conceptos/Llave_Primaria.md):** Garantiza la identidad única de cada registro.
- **[Llave Foránea](../conceptos/Llave_Foránea.md):** Garantiza la coherencia entre tablas, asegurando la integridad referencial.
- **Llave Candidata:** Cualquier atributo o conjunto de atributos que cumple los requisitos para ser PK.
- **Llave Compuesta:** PK formada por la combinación de dos o más atributos.

*Las llaves son la base de la navegación en la [Base de Datos Relacional](../conceptos/Base_de_Datos_Relacional.md).*
