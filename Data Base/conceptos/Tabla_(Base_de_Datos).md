# Tabla (Base de Datos)

Estructura fundamental para organizar información en filas y columnas.

## Componentes
- **[Tupla](../conceptos/Tupla.md) (Registro):** Fila que representa una instancia única de una entidad.
- **[Atributo](../conceptos/Atributo.md) (Columna):** Propiedad o característica de la entidad.

## Integridad
- **[Llave Primaria](../conceptos/Llave_Primaria.md) (PK):** Identificador único del registro; garantiza unicidad y no duplicidad.
- **[Llave Foránea](../conceptos/Llave_Foránea.md) (FK):** Puente entre tablas; referencia a la PK de una tabla "padre".
